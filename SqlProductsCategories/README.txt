В данной папке находятся Sql запросы для реализации задачи:
"В базе данных MS SQL Server есть продукты и категории. 
Одному продукту может соответствовать много категорий, в одной категории может быть много продуктов. 
Напишите SQL запрос для выбора всех пар «Имя продукта – Имя категории». 
Если у продукта нет категорий, то его имя все равно должно выводиться."

Была использована связь многие ко многим. 
Products.sql - sql запрос для создания таблицы с продуктами
Categories.sql - sql запрос для создания таблицы с категориями. 
ProductsCategories.sql - sql запрос для создания связующей таблицы. Товары у которых нет категории попадают в категорию "no category"
SqlSelect.sql - sql запросы (два варианта) для получения пар «Имя продукта – Имя категории».
Example.sql - пример занесения и получения данных
