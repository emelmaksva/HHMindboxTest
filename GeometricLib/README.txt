В данной папке находятся классы библиотеки нахождения площадей фигур.
Shape - абстрактный класс, характерезующий общий функционал фигур (в данном примере это только нахождение площади)
Circle и Triangle - дочерние классы окружности и треугольника соотвественно. 
Triangle содержит в себе дополнительный метод проверки на то, является ли он прямоугольным.
Program - пример использования.
